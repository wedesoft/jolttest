## Chronotest

Simple examples to test the capabilities of Jolt (built under GNU/Linux).

### Dependencies

You need to install [Jolt][1] version 5.1.0.

Further, you need to install [GLFW][3] and [GLEW][4] for visualisation.

### Build

```Shell
make
```
